# Markus

Markus is a neovim-plugin for managing global marks. Built-in methods are probably more viable, this aims to provide a clearer overview and "easier interaction" with marks for people who are new (or noobs) to marks, like myself

## Installation
WARNING: this is for personal use only.


## Usage
Different keymaps provide different functionality. The names these have been given aim to be self describing

- <leader>ms: MarkusShow – Show marks
- <leader>mc: MarkusChose – Chose mark to go to
- <leader>md: MarkusDelete – Delete all marks
- <leader>mx: MarkusSelectDelete – Delete a specific mark
